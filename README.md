# FRAME

## 目录结构
~~~
project             项目工程名
├─app               应用代码文件目录
│  ├─controller     控制器文件
│  ├─model          模型文件
│  ├─view           视图文件
│
├─cache             缓存文件夹
│  ├─logs           日志文件夹
│  ├─...            自定义其他文件夹
│
├─conf              配置目录
│  ├─core.php       核心配置文件
│  ├─app.ini        应用配置文件
│  ├─database.php   数据库配置文件
│  ├─message.php    提示消息配置文件
│  ├─...            自定义其他配置文件
│
├─core              核心目录
│  ├─database       数据库目录
│  ├─utils          有效工具目录
│
├─doc               文档目录
│  ├─...            自定义目录
│
├─webroot           根目录
│  ├─css            样式目录
│  ├─font           字体目录
│  ├─img            文件目录
│  ├─js             脚本目录
│  ├─index.php      入口文件
│
├─README.md         说明文件
~~~
